# ADR-002: Dashboard-first Experience

## Status

Accepted

## Context

Most AI tools are chat-first. AnalyticsOS is intended to become an operating system for professional work, not only a chat interface.

## Decision

The custom dashboard is the primary user experience. Chat is one capability inside the platform.

## Consequences

- The UI must prioritize projects, knowledge, memory, and actions.
- AnythingLLM is infrastructure, not the main interface.
