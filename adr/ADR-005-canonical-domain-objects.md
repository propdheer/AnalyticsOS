# ADR-005: Canonical Domain Objects

## Status

Accepted

## Context

A document-only knowledge base is insufficient for professional intelligence. AnalyticsOS needs to understand real professional objects.

## Decision

AnalyticsOS will model core professional concepts such as Project, Dataset, Business Rule, Decision, Prompt Template, and Knowledge Asset.

## Consequences

- The platform can reason beyond file search.
- APIs and database design become clearer.
- The model must remain simple in early releases.
