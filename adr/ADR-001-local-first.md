# ADR-001: Local-first Architecture

## Status

Accepted

## Context

AnalyticsOS is intended to store professional knowledge, project history, datasets, prompts, and work artifacts. This data may be sensitive.

## Decision

AnalyticsOS will be local-first by default.

## Consequences

- Users retain ownership of their data.
- Cloud integrations must be optional.
- Local installation must remain a first-class experience.
