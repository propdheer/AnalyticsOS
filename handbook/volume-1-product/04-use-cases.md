# Use Cases

## UC-001: Search Prior Work

A user asks:

> Show me everything related to distributor profitability.

AnalyticsOS searches projects, documents, datasets, dashboards, prompts, and memories.

## UC-002: Create a Project Brief

A user asks:

> Create a brief for the RM PM Variance project.

AnalyticsOS combines project metadata, dataset cards, decisions, and related artifacts.

## UC-003: Generate a Prompt for External AI

A user asks:

> Create a Claude prompt to build a Power BI dashboard specification.

AnalyticsOS uses local project context and outputs a structured Markdown prompt.

## UC-004: Remember a Business Rule

A user says:

> Active distributor means billed in the last 60 days.

AnalyticsOS proposes saving this as a Business Rule Card.

## UC-005: Understand a Dataset

A user asks:

> What do we know about the Sales Primary dataset?

AnalyticsOS shows source system, columns, known issues, related projects, and rules.

## UC-006: Build a Management Update

A user asks:

> Draft a weekly leadership update for active analytics projects.

AnalyticsOS uses project timelines, decisions, and recent work to generate an executive update.
