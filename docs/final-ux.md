# Final UX Polish

AnalyticsOS `v0.4.1-alpha` makes the app easier to use by adding a simpler Home screen and grouped navigation.

## UX Principles

1. Start from the user's job, not the system's data model.
2. Show the next best step.
3. Keep advanced capabilities available, but not overwhelming.
4. Make the system loop visible and repeatable.

## New Home Screen

The Home screen has four simple choices:

- Capture context
- Run an action
- Ask your documents
- Preserve work

## Navigation Groups

Sidebar sections are grouped into:

- Start
- Build
- Library
- System

## Recommended Flow

1. Add a Project.
2. Add a Dataset.
3. Add a Business Rule.
4. Run an Action.
5. Save the output as Knowledge.
6. Search or ask the Knowledge Base.
7. Backup.
